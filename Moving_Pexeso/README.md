Pexeso is a memory card game where the player flips over two cards at a time to find matching pairs. The goal is to collect all of the sets in as few rounds as possible.

When we were given an assignment to make this game in WinForms, I thought it was boring. Who plays Pexeso anyway? So I made it better. I made it move. 

In this version of Pexeso, the cards constantly drift around the screen, making it significantly harder to keep track of them and finish the game.
