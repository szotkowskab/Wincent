Being one of my oldest projects, DoN (Dog or Not Bot) is a Discord bot I made for me and my friends. Its main purpose was to play a game, where the user had to guess, whether the image portrayed a dog or something else.

The bot was built using a Discord framework, but the core functional parts and game logic commands were developed entirely by me.

Rather than high quality codes that I would like to show off, this is a project that taught me a lot. It was made in the year 2022, when I was still learning how to code. But it was fun and it was one of the reasons why I kept on programming and got to this point. Maybe it's not the best thing to put into job application, but I am proud I finished it.

Later on I also created a web version of this game. Unfortunately, neither the bot, nor the web version are currently working. The reason is that Discord later changed its attachment hosting policy, which invalidated the image links used in the game.
