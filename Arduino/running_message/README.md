This project allows the user to display a message on their Arduino. The program listens for messages received over the USB serial link and displays them as a continuous right-to-left scrolling text on 4-digit 7-segment display.

It was given to us as an assignment in the Computer Systems subject. The `funshield.h` and `input.h` files were provided by the teachers, so only the `solution.ino` is my own work. 

If you cannot run this program on your own machine, see [this video](https://www.yout-ube.com/watch?v=YCbglNdO1Xo), which visualizes the reference solution, to get a better idea.
